# 💤 Sleep Timer

A lightweight, native Windows 11 utility built with Tauri v2 and React. It allows you to schedule PC shutdown, sleep, or hibernation with an interface that perfectly matches the system design.
##  Features

Native Design: Full integration with Windows 11 Accent Colors and Dark Mode.

Quick Actions: Choose between Shutdown, Sleep, or Hibernate.    

Tray Integration: The app minimizes to the system tray while the timer is running to stay out of your way.

Performance: Extremely low RAM usage thanks to the Rust backend.

##  Installation (Portable)

No installation required.

Download the Sleep-Timer.exe executable from the Releases section.
