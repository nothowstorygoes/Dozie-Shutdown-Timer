## :star2: About the Project

Dozie is a lightweight, modern, and efficient shutdown timer for Windows, built with Tauri v2 and Rust. It is designed for users who want a simple, no-nonsense tool to schedule system power actions without dealing with complex menus or heavy background processes.

### :dart: Features

*   **Power Scheduling:** Easily set a countdown timer to **Shut Down**, **Restart**, or **Hibernate** your PC.
*   **Modern & Intuitive UI:** A clean, user-friendly interface designed for clarity and ease of use.
*   **Background Operation:** Minimize the app to the system tray; it will continue to monitor your countdown without interrupting your workflow.

### :gear: Installation

Dozie is fully portable. This means you don't need to run a heavy installer or deal with registry changes.

*   Navigate to the Releases page.
*   Download the latest .exe file (e.g., Dozie-Shutdown-Timer-v1.1.0.exe).
*   Move the file to any folder you like (e.g., Desktop or Applications).
*   Double-click to run. No installation is required!

In alternative, the app is also available on the [Microsoft Store](https://apps.microsoft.com/detail/9NV1TDXFDZ92?hl=neutral&gl=IT&ocid=pdpshare)

### :eyes: Usage

* Launch the application.
* Set the Timer: Input the number of hours or minutes you wish to wait.
* Select Action: Choose between Shutdown (default), Restart, or Sleep.
* Start: Click the "Start Timer" button. The countdown will continue in background.
